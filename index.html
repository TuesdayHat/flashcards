JavaScript (JS)

JavaScript is the programming language of HTML and the Web.

Operators

JavaScript operators are used to assign values, compare values, perform arithmetic operations, and more.

Variables

Variables are containers that you can store values in. You start by declaring a variable with the var keyword, followed by any name you want to call it.

Variable naming conventions

Variables are named in camel case, in which the first word in the name is not capitalized but any subsequent ones are.

Functions

Functions are a way of packaging functionality that you wish to reuse. When you need the procedure you can call a function, with the function name, instead of rewriting the entire code each time.

Methods

JavaScript methods are the actions that can be performed on objects. A JavaScript method is a property containing a function definition.

Arguments

JavaScript functions have a built-in object called the arguments object. The argument object contains an array of the arguments used when the function was called (invoked).

Parameters

Parameters are the specific variables used in place of the argument when calling a function.

Return

The return statement stops the execution of a function and returns a value from that function.

Chaining methods

Chaining Methods, also known as Cascading, refers to repeatedly calling one method after another on an object, in one continuous line of code.

Strings

A JavaScript string stores a series of characters like "John Doe". A string can be any text inside double or single quotes.

Booleans

JavaScript booleans can have one of two values: true or false.

Undefined

An undefined variable has been declared but has no value.

NaN

Represents a "Not-a-Number" value

Escape

Escaping characters that already have functions within JavaScript allows them to be used normally.

Alert

Alert displays a popup with a specified message and a confirmation button.

Comments

Comments in code are not executed, and are denoted in Javascript by //.

jQuery

jQuery is a widely used pre-existing library of JavaScript commands, similar to what Bootstrap is for CSS.

Attributes

HTML elements have attributes, which are values that affect them and can be changed by JavaScript
